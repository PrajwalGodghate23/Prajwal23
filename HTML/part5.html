Media Tag:-

The HTML <audio> and <video> tags are used to embed audio and video content in a web page, respectively. These tags provide a way to include multimedia elements directly within the HTML document, allowing users to play audio and video files without needing external plugins.

audio tag:

The <audio> tag is used to embed sound content in a web page. It supports various audio formats, including MP3, WAV, and OGG. The <audio> tag can include attributes such as controls, autoplay, loop, and muted to control the playback behavior.

Example:

audio tag code:


<audio controls>
  <source src="audiofile.mp3" type="audio/mpeg">
  <source src="audiofile.ogg" type="audio/ogg">
  Your browser does not support the audio element.
</audio>

video tag:

The <video> tag is used to embed video content in a web page. It supports various video formats, including MP4, WebM, and OGG. Similar to the <audio> tag, the <video> tag can include attributes such as controls, autoplay, loop, width, height, and poster to control the playback behavior and appearance.

Example:

video tag code:


<video width="320" height="240" controls>
  <source src="videofile.mp4" type="video/mp4">
  <source src="videofile.webm" type="video/webm">
  Your browser does not support the video element.
</video>

EMBEDDED VIDEO

You can also embed videos from platforms like YouTube using the <iframe> tag. This allows you to display videos hosted on external sites directly within your web page.

iframes:

The <iframe> tag is used to embed another HTML document within the current document. It creates an inline frame that can display content from another source, such as a different webpage or a video from platforms like YouTube or Vimeo.

Example:

Embedded video code:


<iframe width="560" height="315" src="https://www.youtube.com/embed/vsWxs1tuwDk" title="YouTube video player" ></iframe>

Multi-page Websites:-

A multi-page website is a website that consists of multiple interconnected web pages, each with its own unique URL. These pages are typically linked together through a navigation menu or hyperlinks, allowing users to easily navigate between different sections or topics within the website.

Multi-page websites are commonly used for larger websites that require more content and organization, such as e-commerce sites, blogs, or educational portals.

Example:

Multi-page website code:


<!DOCTYPE html>
<html lang="en">
<head>
 <meta charset="UTF-8">
 <meta name="viewport" content="width=device-width, initial-scale=1.0">
 <title>Home Page</title>
</head>
<body>
<h1>Welcome to My Website</h1>
 <nav>
<ul>
 <li><a href="part 1.html">Introduction to HTML</a></li>
 <li><a href="intro-webdev.html">Introduction to Web Development</a></li>
</ul>
 </nav>
 <p>This is the home page of my multi-page website.</p>
</body>
</html>
