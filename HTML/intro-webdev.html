What is the Internet?

Interconnected network of computer around the world is known as Internet.

How is data transferred?

Data can be transferred from computer A to computer B using physical media like an external hard drive or USB drive, wired connections like a data transfer cable or Ethernet cable, wireless connections like Wi-Fi or Bluetooth, or over the internet via cloud storage, email, or file-sharing services.

What is IP Address?

An IP address (Internet Protocol address) is a unique numerical identifier assigned to each device connected to a computer network that uses the Internet Protocol for communication. It serves two main purposes: identifying the host or network interface and providing the location of the host in the network.

Types of IP Address:-

  • IPv4 (Internet Protocol version 4):- It is the most widely used version of IP addresses. An IPv4 address consists of four sets of numbers separated by periods (.), with each set ranging from 0 to 255.
  • IPv6 (Internet Protocol version 6):- It is the latest version of IP addresses, designed to replace IPv4 due to the limited number of available IPv4 addresses. An IPv6 address consists of eight groups of four hexadecimal digits separated by colons (:).

What is MAC Address?

A MAC (Media Access Control) address is a unique, hardware-based identifier for network interface controllers (NICs) used in local area networks (LANs), such as Wi-Fi and Ethernet. It functions like a physical address or "fingerprint" for devices on a local network, ensuring data packets are routed to the correct destination. MAC addresses are typically a 48-bit value expressed as 12 hexadecimal digits (e.g., 00:1A:2B:3C:4D:5E), with the first three bytes identifying the manufacturer and the last three being a unique identifier

What is Port Number?

A port number is a numerical identifier used in computer networking to specify a particular process or service on a device. It helps direct network traffic to the correct application or service running on a computer or server. Port numbers range from 0 to 65535 and are divided into three categories: well-known ports (0-1023), registered ports (1024-49151), and dynamic/private ports (49152-65535). Common examples include port 80 for HTTP, port 443 for HTTPS, and port 25 for SMTP.

What is DNS - Domain Name System?

The Domain Name System (DNS) is the internet's “phone book” that translates human-readable domain names (like www.google.com) into machine-readable IP addresses (like 74.125.20.147) that computers use to communicate. When you type a website's name into your browser, the DNS system, which involves a series of DNS servers, finds the correct IP address for the website, enabling your browser to connect to the server and load the page.

local area network, Switch & Router

    1. lan area network

      A local area network (LAN) connects devices, like computers and printers, within a small geographic area, such as a home, office, or campus, to enable communication and resource sharing. LANs rely on hardware like routers and switches and technologies such as Ethernet and Wi-Fi for device connectivity and can provide high-speed, private networking for users to share files, printers, and internet access efficiently

      Examples of LANs Home Networks: Connecting your computer, phone, smart TV, and gaming console to share internet and files. Office Networks: Linking office computers to share printers and access central servers for files and applications. Campus Networks: Connecting computers in schools, universities, and libraries to facilitate communication and resource access across the campus.

    2. Switch

      A network switch is a hardware device that connects multiple devices on a local area network (LAN), such as computers and printers, allowing them to communicate by forwarding data packets to their intended destinations. Switches learn the unique MAC address of each connected device and use this information to create a MAC address table, enabling them to send data directly to the correct port, unlike older hub-based networks. Switches operate at the Data Link Layer (Layer 2) of the OSI model but can perform Layer 3 (routing) functions, facilitating traffic management and faster, more efficient data transfer.

    3. Router

      A router is a network device that connects multiple computer networks together, allowing data packets to be sent between them and enabling them to communicate. Routers act as traffic directors, reading packet headers to find the most efficient path to a destination using routing tables. They provide essential functions like sharing a single internet connection, enabling inter-network connectivity, and can offer security features such as firewalls.

Public & Private Address

Public IP addresses are globally unique and connect a network to the internet, while private IP addresses are for internal network use and are not accessible from outside the network. Private IPs are used by devices within a local network, like your home Wi-Fi, and are assigned by the router, whereas public IPs are assigned by an Internet Service Provider (ISP) and are what you see when you search "what is my IP address" on Google. Common private IP ranges include 192.168.x.x, 172.16.x.x to 172.31.x.x, and 10.x.x.x.

What is VPN?

A VPN (Virtual Private Network) creates a secure, encrypted connection over the internet, like a private tunnel for your data. It masks your IP address, encrypts your online activity, and hides your real location, providing greater privacy, security on public Wi-Fi, and the ability to bypass geographic restrictions on content.

What is MERN Stack?

MERN Stack is a popular web development stack that consists of four main technologies: MongoDB, Express.js, React.js, and Node.js. It is used to build full-stack web applications, allowing developers to work with both the frontend and backend using JavaScript.

Who is Full Stack Developer?

A full-stack developer is a software engineer with a broad skill set covering both the front-end (user interface) and back-end (server-side logic, databases, and infrastructure) of a web application or software. They are capable of handling various stages of the software development lifecycle, from building the visible user interface to managing the underlying systems, creating a versatile and valuable asset for tech teams.
