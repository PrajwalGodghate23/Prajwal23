Introduction To HTML

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and structure the content of a webpage.

  • HyperText:This refers to the ability to create links that connect web pages to one another, making the web a connected "web" of information.
  • Markup Language:This means you use "tags" to surround your content, giving that content meaning and structure. You are "marking up" a plain text document.

Creating Your First Webpage

To create a simple webpage, you can use any text editor (like Notepad on Windows or TextEdit on Mac) and save the file with a .html extension. Here's a basic example of an HTML document:


<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
    <h1>Welcome to My Webpage</h1>
    <p>This is a simple HTML document created using a text editor.</p>
    <ul>
        <li>Uses basic tags like <h1>, <p>, and <ul></li>
        <li>No CSS or JavaScriptâ€”just pure HTML</li>
        <li>Great for learning structure and semantics</li>
    </ul>
</body>
</html>

Save this code in a file named index.html and open it in your web browser to see your first webpage!

Basic HTML Tags

  • <!DOCTYPE html>: Declares the document type and version of HTML.
  • <html>: The root element that wraps all the content on the page.
  • <head>: Contains meta-information about the document, like its title and links to stylesheets.
  • <title>: Sets the title of the webpage, which appears in the browser tab.
  • <body>: Contains the main content of the webpage that users see.
  • <h1> to <h6>: Heading tags used to create headings and subheadings. <h1> is the highest level, and <h6> is the lowest.
  • <p>: Defines a paragraph of text.
  • <a href="URL">: Creates a hyperlink to another webpage or resource.
  • <img src="image.jpg" alt="description">: Embeds an image in the webpage. The alt attribute provides alternative text for screen readers.
  • <ul>, <ol>, <li>: Used to create unordered (bulleted) and ordered (numbered) lists. <li> defines each list item.

What is heading?

Headings are used to create titles and subtitles on a webpage. They help organize content and make it easier for users to read and navigate. HTML provides six levels of headings, from <h1> to <h6>, with <h1> being the most important and typically used for main titles, while <h6> is the least important and used for sub-subtitles.

Example of headings:


    <!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>
<h3>This is a Heading 3</h3>
<h4>This is a Heading 4</h4>
<h5>This is a Heading 5</h5>
<h6>This is a Heading 6</h6>
</body>
</html>

output.

This is a Heading 1

This is a Heading 2

This is a Heading 3

This is a Heading 4

This is a Heading 5
This is a Heading 6

What is paragraph?

A paragraph in HTML is defined using the <p> tag. It is used to group sentences and provide structure to the text content on a webpage. Paragraphs help improve readability by breaking up large blocks of text into smaller, manageable sections.

Example of a paragraph:


<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
<p>This is a simple paragraph of text on my webpage. It provides information and
helps organize content for better readability.</p>
</body>
</html>

output.

This is a simple paragraph of text on my webpage. It provides information and helps organize content for better readability.

What is hyperlink?

A hyperlink, commonly referred to as a link, is an element in HTML that allows users to navigate from one webpage to another or to different sections within the same page. Hyperlinks are created using the <a> (anchor) tag, which includes an href attribute that specifies the destination URL of the link.

Example of a hyperlink:


<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
<a href="https://www.example.com">Click here to visit Example.com</a>
</body>
</html>

output.

Click here to visit Example.com

What is an Element?

An element in HTML is a fundamental building block of a webpage. It consists of a start tag, content, and an end tag. Elements are used to structure and format the content on a webpage, defining how different parts of the content should be displayed and organized.

Example of an HTML element:


<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
<h1>This is a Heading</h1> <
<p>This is a paragraph of text.</p>
</body>
</html>

output

This is a Heading

This is a paragraph of text.

What is Attribute?

Attributes in HTML provide additional information about an element and are included within the start tag. They consist of a name and a value, separated by an equals sign. Attributes help define the behavior, appearance, or functionality of an element.

Example of an HTML element with attributes:


<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
<a href="https://www.example.com" target="_blank">Visit Example.com</a>
<img src="assets/mern.png" alt="A descriptive text">
</body>
</html>

output

Visit Example.com

What is an Image?

An image in HTML is embedded using the <img> tag, which is a self-closing tag that does not require an end tag. The <img> tag includes attributes such as src (source) and alt (alternative text) to specify the image file's location and provide descriptive text for accessibility.

Example of embedding an image in HTML:


<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
<img src="assets/mern.png" alt="A descriptive text">
</body>
</html>

output

What is List?

A list in HTML is a way to organize and present related items in a structured format. HTML provides two main types of lists: unordered lists and ordered lists. Unordered lists are used for items that do not have a specific order, while ordered lists are used for items that follow a sequence or ranking.

Example of an unordered list:


<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
<h2>My Favorite Fruits</h2>
<ul>
    <li>Apple</li>
    <li>Banana</li>
    <li>Cherry</li>
</ul>
</body>
</html>

output

My Favorite Fruits

  • Apple
  • Banana
  • Cherry

Example of ordered lists:


<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
<h2>My Favorite Fruits</h2>
<ol>
    <li>Apple</li>
    <li>Banana</li>
    <li>Cherry</li>
</ol>
</body>
</html>

output

My Favorite Fruits

  1. Apple
  2. Banana
  3. Cherry
