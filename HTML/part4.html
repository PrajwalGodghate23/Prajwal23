FORMS in HTML

FORMS:

HTML forms, defined using the

Tags are essential for collecting user input on web pages. They incorporate a variety of interactive controls such as text fields, numeric inputs, email fields, password fields, checkboxes, radio buttons, and submit buttons. Over 85% of websites rely on forms to gather data from users, making them a fundamental component of modern web development.


<!DOCTYPE html>
<html>
<head>
    <title>Basic HTML Form</title>
</head>
<body>
    <h2>Contact Form</h2>
    <form action="submit.html" method="post">
        <label for="name">Name:</label>
        <input type="text" id="name" name="name"><br><br>

        <label for="email">Email:</label>
        <input type="email" id="email" name="email"><br><br>

        <input type="submit" value="Submit">
    </form>
</body>
</html>

OUTPUT:

Contact Form





Radio Buttons & Checkboxes

Radio buttons and checkboxes are HTML input elements used to gather user selections in forms. The key difference is that radio buttons allow a user to make only a single choice from a group, while checkboxes permit multiple selections.

Example


<!DOCTYPE html>
<html>
<head>
    <title>Radio and Checkbox Example</title>
</head>
<body>
    <h2>Choose Your Gender</h2>
    <form>
        <input type="radio" id="male" name="gender" value="male">
        <label for="male">Male</label><br>

        <input type="radio" id="female" name="gender" value="female">
        <label for="female">Female</label><br>

        <input type="radio" id="other" name="gender" value="other">
        <label for="other">Other</label>
    </form>

    <h2>Select Your Hobbies</h2>
    <form>
        <input type="checkbox" id="reading" name="hobby" value="reading">
        <label for="reading">Reading</label><br>

        <input type="checkbox" id="gaming" name="hobby" value="gaming">
        <label for="gaming">Gaming</label><br>

        <input type="checkbox" id="traveling" name="hobby" value="traveling">
        <label for="traveling">Traveling</label>
    </form>
</body>
</html>

output:

Choose Your Gender



Select Your Hobbies




Place holder attribute:

The placeholder attribute specifies a short hint that describes the expected value of an input field (e.g. a sample value or a short description of the expected format).

example


<!DOCTYPE html>
<html>
<head>
    <title>Placeholder Example</title>
</head>
<body>
    <h2>Login Form</h2>
    <form>
        <input type="text" placeholder="Enter your username"><br><br>
        <input type="password" placeholder="Enter your password"><br><br>
        <input type="submit" value="Login">
    </form>
</body>
</html>

output:

Login Form
