Nested Lists & Tables

Nested lists :

A nested list in HTML is a list that contains other lists within its list items, creating a hierarchical structure. This allows for organizing information into categories and subcategories, providing a clear visual representation of relationships between items.

Creating Nested Lists:


<ul>
  <li>India
    <ul>
      <li>Nagpur</li>
      <li>Mumbai</li>
      <li>Delhi</li>
    </ul>
  </li>
  <li>USA
    <ul>
      <li>New York</li>
      <li>Los Angeles</li>
      <li>Chicago</li>
    </ul>
  </li>
  <li>Japan
    <ul>
      <li>Tokyo</li>
      <li>Osaka</li>
      <li>Kyoto</li>
    </ul>
  </li>
</ul>

    • Tables in Html:

    HTML tables are used to display data in a structured, tabular format, arranged in rows and columns. They are ideal for presenting data where a clear relationship exists between the cells, such as product lists with prices, employee records with salaries, or flight information.

    Here are the key elements for creating HTML tables: <table>

    Here are the key elements for creating HTML tables: <table>: This tag defines the entire table and acts as its container. <tr>: This tag defines a table row. Each <tr> element represents a new row within the table. <th>: This tag defines a table header cell. <th> elements are typically used within the first <tr> to label the columns and are usually rendered in bold by default. <td>: This tag defines a standard table data cell. <td> elements contain the actual data within each row.

    example

    
<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" 
          content="width=device-width, initial-scale=1.0">
    <title>HTML</title>
</head>
<body>
    <table>
        <tr>
            <th>Firstname</th>
            <th>Lastname</th>
            <th>Age</th>
        </tr>
        <tr>
            <td>Priya</td>
            <td>Sharma</td>
            <td>24</td>
        </tr>
        <tr>
            <td>Arun</td>
            <td>Singh</td>
            <td>32</td>
        </tr>
        <tr>
            <td>Sam</td>
            <td>Watson</td>
            <td>41</td>
        </tr>
    </table>
</body>
</html>

    output:

    HTML
    Firstname Lastname Age
    Priya Sharma 24
    Arun Singh 32
    Sam Watson 41
